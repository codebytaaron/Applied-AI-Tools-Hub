# AI Resume Analyzer

A full-stack resume analysis platform designed to evaluate resumes the way recruiters and ATS systems do.

This tool processes a PDF resume and returns a structured analysis covering impact, structure, keyword alignment, and risk areas, with clear guidance on what to improve first.

Built as a practical, no-frills system focused on signal, not hype.

No accounts. No tracking. No external AI APIs.
