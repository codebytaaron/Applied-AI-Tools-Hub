# cpp-safety-checklist

C++ CLI template that generates a safety checklist from job type.

This repo is intentionally written in **Cpp** (not Python/JS) so GitHub shows a different language mix.

## Run
```bash
g++ -std=c++17 main.cpp -o app
./app
```

## Input format
Put your notes in `input.txt` (or pipe stdin). Output is printed to stdout.
