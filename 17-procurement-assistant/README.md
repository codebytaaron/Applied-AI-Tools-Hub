# Procurement Comparison Assistant

Compares vendor quotes and recommends a winner based on weighted criteria.

## What it does
Given quotes and criteria, outputs a comparison matrix, risks, negotiation points, and a recommendation.

## Quick start
```bash
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt

cp .env.example .env
# set MOCK_MODE=true to run without an API key
python main.py --in examples/input.txt --out out.md
```

## Customize
- Edit `prompt.py` to fit your rules, formatting, and required outputs.
- Add real examples to `examples/`.

## Output
- Writes a structured result to `out.md`.
