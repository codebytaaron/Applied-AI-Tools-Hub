## Overview

AI Text Quality Evaluator reviews written content and provides a structured assessment of:

- Clarity
- Tone
- Readability
- Overall writing quality

The model is designed to give direct, understandable feedback without rewriting the original text.

---

## Features

- Word and sentence analysis
- Average sentence length calculation
- Clarity classification
- Tone detection
- Overall quality score
- Actionable feedback
